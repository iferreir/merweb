Welcome to MERRILL

MERRILL is an open source 3D micromagnetics software package with a simple scripting user interface that requires little computational knowledge from the user in order to model complex inhomogeous domain structures in magnetic materials.

MERRILL uses the finite-element/boundary-element numerical method best suited for modelling irregular grain geometries that are of interest to the rock and paleomagnetic community. It is able to simulate a range of magnetic characteristics of both single grains and small assemblies of interacting grains. The open source nature of the code encourages future development of the model by the scientific community.
