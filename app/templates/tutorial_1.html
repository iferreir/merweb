MERRILL Tutorial 1 – Download, install, and run your first model

This tutorial is an introduction to using MERRILL. It shows you how to download the software, install it, and run your first simple micromagnetic model.

1. Watch the tutorial video

The video is based on the original Tutorial 1 from the old MERRILL website.

2. What you will learn

  • Where to find and download the latest MERRILL executables.
  • How to install MERRILL on your computer.
  • How to open a terminal or command prompt and run MERRILL.
  • How to run a simple example model shown in the video.
  • Where the output files are written and how to check the run worked.

Later tutorials will give detailed written, step‑by‑step instructions for Windows, macOS, and Linux.
