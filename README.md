# Merrill-Web: FastAPI Scientific Web Interface
A modern, fast, and easy-to-use web interface for the MERRILL micromagnetic modeling software, built with FastAPI.

Merrill-Web provides a simple site that surfaces downloads, tutorials, and project background for the MERRILL ecosystem. It uses FastAPI with Jinja2 templates and Bootstrap 5 for a clean, responsive UI. Static assets are bundled locally under static/ for offline-friendly development.
